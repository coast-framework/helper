# helper
Helper functions for the pragmatic clojurian

## Installation

Add this to your [deps.edn](https://clojure.org/guides/deps_and_cli)

```clojure
coast-framework/helper {:mvn/version "1.0.0"}
```

## Usage

Require it like this

